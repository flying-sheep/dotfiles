dotfiles
========

Managed using toml-bombadil_.

.. code:: nu

  bombadil link
  bombadil watch

Needs the following packages:

- ``carapace`` (called ``carapace-bin`` on Arch Linux)
- ``bat``, ``exa``, ``ripgrep``

On Linux also:

- ``ksshaskpass``
- ``pkgfile``

.. _toml-bombadil: https://oknozor.github.io/toml-bombadil/
